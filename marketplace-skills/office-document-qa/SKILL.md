# Office Document QA

## Workflow
1. Validate document structure: objective, scope, timeline, owner, dependencies.
2. Detect ambiguous wording, inconsistent terms, and unsupported claims.
3. Suggest precise rewrites for critical paragraphs.
4. Return a pre-send checklist with high-priority fixes first.

## Priority Rules
- Critical: changes that alter meaning or commitment.
- Important: clarity and consistency issues.
- Nice-to-have: style polish.
